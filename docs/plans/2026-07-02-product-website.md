# UGK 产品官网计划

## 目标

把 `https://ugk-task-share.pages.dev/` 从“直接进入 Task 市场”改成面向普通用户的产品官网，让第一次接触 UGK 的用户先理解：

- UGK 是能按自然语言执行电脑任务的 AI 工具。
- 不会写代码也能从 Task 市场直接使用别人做好的自动化。
- 市场仍是核心入口，但不再承担产品解释。

## 路由

- `/`：产品官网。
- `/marketplace/`：现有 Task 分享市场。
- `/upload/`、`/account/`、`/admin/`、`/cli-auth/`：保留现有路径。

## 页面结构

参考成熟产品页的信息节奏，首页不再按普通说明书排卡片，而是按“承诺 -> 工作流 -> 市场 -> 任务类型 -> 能力底座 -> FAQ/CTA”推进。

1. 首屏：明确承诺“不用会编程，也能让电脑替你跑任务”，右侧使用真实 UGK 终端产品截图。
2. 工作流：解释用户只需要说目标、选 Task、看执行。
3. 市场入口：用真实市场截图承接“现成 Task 可安装”。
4. 可交出去的任务：整理归档、网页检查、批量处理、报告生成、复用流程。
5. 能力底座：上下文、工具执行、用户控制。
6. FAQ 和最终 CTA：回答小白疑问，引导打开 Task 市场。

## 设计约束

- 复用当前 Task 市场的深色、绿色高亮、终端感和按钮样式。
- 不引入新框架、不新增构建流程。
- 首屏使用现有市场截图作为真实产品信号。
- 正文先做中文消费者版本，完整多语言后续再补。

## 验收

- 根路径能看到产品官网。
- `/marketplace/` 能看到原市场和筛选控件。
- 其它页面导航到市场时不再指向根页锚点。
- 现有 task-share 测试通过。
