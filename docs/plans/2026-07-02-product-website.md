# UGK 产品官网计划

## 目标

把 `https://ugk-task-share.pages.dev/` 从“直接进入 Task 市场”改成面向普通用户的产品官网，让第一次接触 UGK 的用户先理解：

- UGK 是能按自然语言执行电脑任务的 AI 工具。
- 不会写代码也能从 Task 市场直接使用别人做好的自动化。
- 市场仍是核心入口，但不再承担产品解释。

## 路由

- `/`：产品官网。
- `/marketplace/`：现有 Task 分享市场。
- `/upload/`、`/account/`、`/admin/`、`/cli-auth/`：保留现有路径。

## 页面结构

1. 首屏：一句话解释产品，主按钮指向开始使用，次按钮指向 Task 市场。
2. 使用场景：整理文件、收集资料、批量处理、网页操作。
3. 三步开始：找 Task、描述目标、确认执行。
4. 市场入口：解释 Task 市场是现成自动化库。
5. 信任区：本地执行、确认后运行、任务可审查。
6. FAQ：回答小白最常问的问题。

## 设计约束

- 复用当前 Task 市场的深色、绿色高亮、终端感和按钮样式。
- 不引入新框架、不新增构建流程。
- 首屏使用现有市场截图作为真实产品信号。
- 正文先做中文消费者版本，完整多语言后续再补。

## 验收

- 根路径能看到产品官网。
- `/marketplace/` 能看到原市场和筛选控件。
- 其它页面导航到市场时不再指向根页锚点。
- 现有 task-share 测试通过。
