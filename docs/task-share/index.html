FAQ

常见问题

我不会写代码，可以用吗？

可以。你可以先从 Task 市场安装现成任务，再用自然语言描述要处理的内容。

Task 市场是什么？

它是 UGK 的自动化任务库，里面的 Task 可以被安装到本地 UGK 使用。

它会直接替我乱点吗？

不会。涉及登录、发布、授权等关键动作时，流程会保留确认。
