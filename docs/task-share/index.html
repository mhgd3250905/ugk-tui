Install

三种安装方式

新手用安装器，熟悉 npm 或源码安装的用户可以选择对应渠道。

自动检测 Node/npm，安装 UGK，并引导配置 DeepSeek API key。

# 推荐给第一次安装 UGK 的用户
$ npx ugk-install

已安装 Node.js 18+ 时，直接全局安装后运行。

# 已有 Node.js 18+ / npm 环境
$ npm i -g ugk-agent
$ ugk

想从源码跟进最新版时，直接 clone 仓库后本地链接。

# 从 GitHub 克隆源码并链接本地命令
$ git clone https://github.com/mhgd3250905/ugk-tui.git
$ cd ugk-tui
$ npm install
$ npm link
$ ugk

FAQ

常见问题

我不会写代码，可以用吗？

可以。你可以先从 Task 市场安装现成任务，再用自然语言描述要处理的内容。

Task 市场是什么？

它是 UGK 的自动化任务库，里面的 Task 可以安装到本地 UGK 使用。

它会直接替我乱点吗？

不会。涉及登录、发布、授权等关键动作时，流程会保留确认。
